### jupyter Notenood명령어

```commandline
명령모드( 활성화 하려면 Esc 키를 누름)
Shift-Enter : 셀 실행, 아래 선택
Ctrl-Enter : 셀 실행
Alt-Enter : 셀 실행, 아래에 삽입
y : 코딩
m : 마크다운으로
a : 위에 셀 삽입
b : 아래에 셀 삽입
x : 선택한 셀 잘라내기
c : 선택한 셀 복사
Shift-V : 위 셀 붙여넣기
V : 아래에 셀 붙여넣기
Z : 마지막 셀 삭제 취소
D,D : 선택한 셀 삭제
Shift-M : 아래 셀 병합

편집 모드( 활성화 하려면 Enter키 누름)
Shift-Tap : 툴팁표시
Ctrl-] : 들여쓰기
Ctrl-Shift : 셀 분할


```
